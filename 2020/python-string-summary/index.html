Python字符串截取总结

Written by Jessica with ♥ on March 10, 2020 in 编程语言

今天要处理python的字符串，总结一下罗列如下： str = ‘0123456789’ str[5]: 这个和list列表找索引一样，结果是【5】 str[1:5:]: 结果是：【1,2,3,4】,后面最后一个参数可以省略，故也可以写成:str[1:5] str[1👎2]:从字符串下标1开始，到最后一位（不包括最后一位),步长是2 str[-6::-1]:步长小于0，那么截取位置就是原对象从后往前取。取数（结果）的时候也是从后往前取得【4,3,2,1,0】 str[:-2:-1]:step<0,从后往前截取，然后取得的结果从后往前写，结果：【9】正向截取不到就反向取 str[-2:2:-2]:step<0 从后往前截取，然后倒着写出结果【8,7,6,5,4,3】，然后加上step，就是从8开始减去step=2，最终结果【8,6,4】
